# 下载代码

```shell
git clone https://github.com/tiandizhiguai/dhorse.git
```

# Java11及以上版本

在代码主目录执行如下命令：

```shell
mvn clean install
```

# Java8版本

* 修改主目录下的pom.xml文件

1. 13-14行改为：

```xml
<!--<java.version>11</java.version>-->
<java.version>1.8</java.version>
```

2. 179-183行改为：

```xml
<!--<release>${java.version}</release>-->
<source>${java.version}</source>
<target>${java.version}</target>
```

* 在代码主目录执行如下命令：

```shell
mvn clean install
```
