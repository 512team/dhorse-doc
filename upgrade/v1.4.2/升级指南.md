## 升级步骤

##### 下载安装文件

```shell
wget https://gitee.com/i512team/dhorse/releases/download/v1.4.2/dhorse-v1.4.2-bin.tar.gz
```

##### 解压

```shell
tar zxvf dhorse-v1.4.2-bin.tar.gz
```

##### 进入安装目录

```shell
cd dhorse-v1.4.2
```

##### 修改配置

```shell
vim conf/dhorse.yml
data.path=旧版本的值
```

##### 重启服务

```
bin/dhorse-stop.sh
bin/dhorse-start.sh
```
